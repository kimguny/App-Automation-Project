# Project Description
### App Automation Project
Baedal Minjok App UI Test

### Explanation
이 프로그램은 휴대폰을 연결하여 배달의 민족 앱 자동화 테스트를 하는 프로그램입니다. 
배달의 민족 앱 UI가 바뀌었을 경우 코드를 다시 작성해줘야 합니다. 

### Scenario
- 어플 실행
- 배민, 치킨, 배달 빠른순, 초기화 순으로 클릭
- 좌→우 스크롤
- 뒤로가기 2번
- 이삭토스트 검색
- 맨위 항목 클릭
- 아래로 스크롤 2번
- 커피 클릭
- H.아메리카노 1개 담기
- 뒤로가기 3번, 메인화면에서 앱 종료
